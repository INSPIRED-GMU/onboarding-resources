## Installing

Make sure you [install the font](https://github.com/tonsky/FiraCode/wiki/Installing).

## Enabling

**Preferences** --> **Settings**

**Add** before `"ignored_packages":`

```
"font_face": "Fira Code",
"font_options":
[
	"subpixel_antialias"
],
```

## Antialias

If you want enable antialias, add in font_options: `"gray_antialias"`
